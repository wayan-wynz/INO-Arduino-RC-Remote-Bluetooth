# Arduino-RC-Remote-Bluetooth
RC Car/Tank menggunakan Mikorkontroler Arduino Uno. <br>
**Catu daya menggunakan  Baterai Lipo 2S (7,4v)** 

## Bahan-bahan Utama
- Arduino Uno
- Modul Bluetooth Hc-05
- Motor Driver L298N
- Lampu LED
